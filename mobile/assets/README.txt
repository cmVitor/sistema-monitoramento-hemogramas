Pasta de Assets do App Mobile

Esta pasta deve conter os seguintes arquivos de imagem:

1. icon.png (1024x1024px)
   - Ícone principal do app
   - Usado na App Store/Play Store

2. adaptive-icon.png (1024x1024px)
   - Ícone adaptativo para Android
   - Área segura no centro de 512x512px

3. splash.png (1284x2778px)
   - Tela de splash (carregamento)
   - Fundo branco com logo centralizado

4. notification-icon.png (96x96px)
   - Ícone pequeno para notificações Android
   - Deve ser monocromático (branco com transparência)

Você pode gerar esses assets usando:
- Figma (https://figma.com)
- Canva (https://canva.com)
- App Icon Generator (https://appicon.co)

Ou usar assets temporários do Expo:
- Execute: npx expo install expo-splash-screen
- Expo gerará assets padrão automaticamente
